Windows下如何快速的将图片上传至图库并插入markdown

工具

  • markdown编辑器：Typora
  • 图床：postimages
  • 截图及图片上传工具：Postimage

方法

Typora是在Windows下算是相当好用的markdown编辑器，但是其在图片上传方面还是不太方便。
在Mac平台下，Typora是可以自动将插入的本地图片上传至云，并将markdown中的链接进行转换的。可惜暂时（2018/8/21），Linux和Windows版本都还没有此功能。

不过我们可以使用postimages提供的工具来达到相似的效果。

值得注意的是，postimages疑似被墙，无法在国内打开。但是，不影响上传至postimages的图片。
这些图片还是可以在国内正常浏览。
